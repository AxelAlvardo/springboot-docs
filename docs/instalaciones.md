# Instalación de Spring Boot

## Requisitos

| Herramienta   | Versión recomendada |
| ------------- | ------------------- |
| Java          | 21                  |
| Maven         | 3.9+                |
| IntelliJ IDEA | Última versión      |
| Spring Boot   | 3.x                 |

!!! warning "Importante"
Verifica que Java esté correctamente configurado en la variable PATH.

## Instalación de Java

```bash
java -version
```
